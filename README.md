# React + TypeScript + Vite + shadcn/ui

This is a template for a new Vite project with React, TypeScript, and shadcn/ui.

## Adding components

To add components to your app, run the following command:

```bash
npx shadcn@latest add button
```

This will place the ui components in the `src/components` directory.

## Using components

To use the components in your app, import them as follows:

```tsx
import { Button } from "@/components/ui/button"
```

TheoChemMerida
The TheoChemMerida Research Group is part of the Applied Physics Department at the Centro de Investigación y de Estudios Avanzados (CINVESTAV), Unidad Mérida. The group is led by Dr. Gabriel Merino and Dr. Filiberto Ortiz Chi, and focuses on the use of modern computational chemistry tools to understand and predict the properties of novel organic and inorganic molecular systems.

Our research interests include planar hypercoordinate carbon molecules, systems containing multiple bonds, endohedral complexes, dimetallocenes, complex materials, and molecules under extreme conditions such as high pressure. A central objective of our work is to gain fundamental insight into the structure, bonding, and electron delocalization of these systems.
